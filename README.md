# distincter2

A tool that can check lists in Markdown files and find duplicates.

## How to use

### Global installation

```shell
$ gem i distincter2
$ distincter2 ./path_to_directory_with_markdown_files
```

### Local installation

```shell
$ ./bin/local_distincter2 ./path_to_directory_with_markdown_files
```
